Final Reflection

WHAT: Key Experiences and Learning Outcomes

Hands-on activities in this module helped me learn how AI and machine learning are changing businesses as part of the Fourth Industrial Revolution. Working on a Convolutional Neural Network (CNN) for picture classification with the CIFAR-10 dataset was a key experience.

This was my first time working with CNNs, and it was both hard and fun to try to make them work like real neural networks. I learnt a lot about machine learning by setting up neural layers, pooling layers, and backpropagation to make the model work better.

SO WHAT: Interpretation and Impact

These events have had a huge impact on how I think about the moral, economic, and social effects of machine learning. It's become clear to me that fairness, transparency, and responsibility are just as important as accuracy in models.

For example, the biassed datasets we used for our Unit 6 group project made me realise how easy it is for machine learning models to reinforce social biases if they are not used properly. As part of our project, I helped find and fix these kinds of flaws by preprocessing the data and choosing the right features.

NOW WHAT: Future Applications and Growth

I'm going to use the things I learnt in this lesson on bigger and more complicated machine learning projects in the future. I want to learn more about unsupervised learning and reinforcement learning in particular because I think they have a lot of promise for real-time data analysis, especially in finance and healthcare.

Another thing I'm working on is getting better at choosing models and judging them. In order to make models that are useful in the real world, I want to make sure that they are not only correct but also open, honest, and moral.
